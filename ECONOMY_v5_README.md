📊 МЕХАНИКА ДОХОДОВ:

1. ФЕРМЫ ДАЮТ ДОХОД РАЗА В ЧАС:
   • Используй !harvest каждый час
   • Каждая ферма даёт X монет в час
   • Можно иметь несколько ферм одновременно
   • Доход складывается!

2. БОНУСЫ ГИЛЬДИИ (апгрейды):
   • +5-15% к доходу членов от ферм
   • +5-15% к взносам в казну
   • Специальные эффекты (тайные бонусы за удачу)

3. КАЗНА ГИЛЬДИИ:
   • От ферм членов часть идёт в казну (с бонусом)
   • Лидер может тратить из казны
   • Больше апгрейдов = больше пассивного дохода коллектива
